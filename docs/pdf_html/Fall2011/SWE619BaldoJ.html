Adjunct Professor: James Baldo Jr.

Brief Course Description:

The purpose of this course is to provide the student with an overall understanding of the following state-of-the-practice software construction areas:

·         constructing sequential and concurrent programs;

·         constructing software components based on control and packaging structures;

·         constructing software components based on concepts such as information hiding, data abstraction, procedural, inheritance, design patterns, polymorphism, iteration, and object-based/object-oriented principles;

·         constructing software components of high quality; and

·         constructing software components based on exception handling.
