Course Description

Concepts and techniques in data mining and multidisciplinary applications. Topics include databases; data cleaning and transformation; concept description; association and correlation rules; data classification and predictive modeling; performance analysis and scalability; data mining in advanced database systems, including text, audio, and images; and emerging themes and future challenges.

Prerequisites
  • Formally: Grade of C or better in CS 310 (Data Structures) and STAT 344 (Probability and Statistics) or equivalent.

  • More specifically: Programming experience in Python preferred. Experience in Java or C++ will work as well, but the assignments will use the Python framework. Students should be familiar with basic probability and statistics concepts, and linear algebra. Please expect lots of programming in all the assignments.
Grading

Programming Assignments: 40%
Midterm: 20%
Final Exam: 30%
Class participation/Activities/quizzes: 10%
          Extra credit: competition winners for homework
Assignments

There will be 4-5 competition-style programming assignments. The preferred programming language is Python. Competition winners will get 1% extra credit added to the final grade.

Exams

There will be one midterm and one final exam. The final exam is comprehensive. Both exams are closed-book, and they must be taken at the scheduled time and place, unless prior arrangement has been made with the instructor. Missed exams cannot be made up.

Class Participation

You will be able to earn class participation credit through in-class activities and quizzes. The purpose of the quizzes is to help you stay on track of the lecture materials, so they are typically short and easier compared to the midterm and final.
