About the Class

This course is a requirement of the ACS Game Design degree and can be used as a Senior CS Elective for the BS CS degree.

The course will provide an introduction to technologies and techniques used in modern computer games, animations, and special effects. Students will gain knowledge and experience needed to build games, simulations, and animations. Therefore, the ey elements of the course will be knowledge, experience, and fun!

Topics are likely to include:

  • C++ (Expect to gain a lot of experience in C++ programming and debugging)
  • Scripting (Python and/or Lua)
  • Asset creation, modification, conversion, and management
  • Path finding, navigation, and steering behaviors
  • Physics: particle simulations, collision detection and response, dynamics
  • 3D Animation tools: Hierarchical animation, FK and IK, Keyframe, interpolation, motion capture, quaternions, parametric blending techniques, modeling physical and articulated objects, forward and inverse kinematics, key-frame, procedural, and behavioral animation, and free-form deformation.
  • Camera control, fly-through, splines
  • Game AI: state machines, rules, fuzzy logic, sensing, memory
  • Event handling and input devices
  • Level saving (serialization)

Course Outcomes

a) Become familiar with advanced techniques used in object-oriented programming

b) Realize the manner with which the graphics pipeline may be utilized to create animations, games, and simulations

c) Utilize mathematical abilities to create optimum algorithms

d) Exhibit skills that demonstrate understanding of physics simulations

e) Realize the appropriate techniques for maximizing code reuse

f) Experience employing a variety of data structures and algorithms
