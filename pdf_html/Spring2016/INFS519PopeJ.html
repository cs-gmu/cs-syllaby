COURSE DESCRIPTION

INFS 519 is designed to introduce students to the following topics:

  • Dynamic lists
  • Linked lists
  • Stacks and queues
  • Trees
  • Graphs
  • Generics
  • Recursion
  • Searching/Sorting
  • Hashing

We may also cover topics not listed above if time allows and there is sufficient interest.
