Course Content

Study of computational models of visual perception and their implementation in computer systems. Topics include:
  • Image formation, mathematical and physical background
  • Image representation and data structures for image analysis
  • Image preprocessing
  • Segmentation
  • Shape representation and description
  • Object recognition
  • Image understanding
  • 3D vision, geometry
  • Motion analysis
Applications to
  • video surveillance
  • human motion analysis
  • gesture recognition and human-computer interaction
  • intelligent highways
are presented.
