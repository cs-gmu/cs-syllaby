Class Description: Current and emerging issues in advanced computer networks and applications. Topics include high speed networking architectures and protocols, mobile ad hoc networks, sensor systems, delay tolerant networking and emerging cyber-physical systems.

This class is taught as a research seminar. The instructor will present several papers during the first few weeks, after which students will take turns presenting papers. Each student will be responsible for a semester-long project. The project will be of the student's choosing, in conjunction with the instructor. There will be no exams for this class. Grades are determined by class participation, class presentation and the final project.

Papers and all other course materials are available at the course web site on blackboard.
